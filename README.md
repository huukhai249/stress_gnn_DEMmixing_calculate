# stress_gnn_DEMmixing_calculate

## Description
This repository contains a collection of Python modules designed for calculating the stress in granular materials using Graph Neural Networks (GNN) in the context of Discrete Element Method (DEM) mixing calculations. It aims to facilitate simulations and analyses in computational mechanics.

## Installation
To install the package, clone the repository and install the required packages:

```bash
git clone https://github.com/huukhai249/stress_gnn_DEMmixing_calculate.git
cd stress_gnn_DEMmixing_calculate
pip install -r requirements.txt
